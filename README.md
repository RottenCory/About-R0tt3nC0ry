# My Personal Website

A dark nature-themed personal website to share your identity, personality, and interests.

## Pages

- **Home** (`index.html`) - Welcome page
- **About Me** (`about.html`) - Your story, identity, pronouns, diagnoses
- **Personality** (`personality.html`) - How you see yourself, strengths, values
- **Interests** (`interests.html`) - Hobbies, media, communities
- **Contact** (`contact.html`) - How to reach you

## How to Edit

1. Open any `.html` file on GitHub
2. Click the pencil icon to edit
3. Look for comments like `<!-- ADD YOUR CONTENT HERE -->`
4. Replace the placeholder text with YOUR information
5. Scroll down and click "Commit changes"

## Color Theme

- **Primary Dark**: Black
- **Secondary**: Dark Gray
- **Accent**: Muted Forest Green
- **Text**: Off-white
